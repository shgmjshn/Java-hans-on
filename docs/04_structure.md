# 第4章：Javaプログラムの構造  
―「読めるコード」を書く ―

---

## この章のゴール 🎯

- Javaコードの構造を理解できる
- `{}` の意味が分かる
- コメントの正しい使い方を知る

---

## 1. Javaはブロック構造

```java
{
    // この中が1つのブロック
}
```

- クラス
- メソッド
- 制御構文

すべて `{}` で表現されます。

---

## 2. クラスのブロック

```java
class Sample {
}
```

👉 **Javaは必ずクラスの中** に書きます。

---

## 3. メソッドのブロック

```java
void method() {
}
```

👉 **処理のまとまり** です。

---

## 4. インデントの重要性

```java
class A {
    void m() {
        System.out.println("OK");
    }
}
```

- 動作は変わらない
- **可読性が大きく変わる**

---

## 5. コメントの種類

### 1行コメント

```java
// コメント
```

### 複数行コメント

```java
/*
 コメント
*/
```

### ドキュメントコメント

```java
/**
 * 説明
 */
```
