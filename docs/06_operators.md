この章を終えると、次のことができるようになります。

- 条件判断の仕組みがわかる

---

## 1. 算術演算子

| 演算子 | 意味 |
|---|---|
| `+` | 加算 |
| `-` | 減算 |
| `*` | 乗算 |
| `/` | 除算 |
| `%` | 余り |

```java
int a = 10;
int b = 3;
System.out.println(a / b); // 3
```

👉 **整数同士の割り算は整数。**

---

## 2. 代入演算子

```java
int x = 5;
x += 3; // x = x + 3 と同じ
```

---

## 3. 比較演算子

```java
int age = 20;
System.out.println(age >= 18); // true
```

---

## 4. 論理演算子

```java
boolean adult = true;
boolean ticket = false;

System.out.println(adult && ticket); // false
```

- `&&` : かつ (AND)
- `||` : または (OR)
- `!` : 否定 (NOT)

---

## 5. `==` と `equals` の違い

### プリミティブ型

```java
int a = 10;
int b = 10;
System.out.println(a == b); // true
```

### 参照型（String）

```java
String s1 = "Java";
String s2 = "Java";

System.out.println(s1 == s2);      // 参照比較（同じ場所か？）
System.out.println(s1.equals(s2)); // 中身比較（同じ文字か？）
```

👉 **文字列比較は必ず `equals` を使う**
